# Student Performance Predictor 🎓

Machine Learning model that predicts student GradeClass based on lifestyle and academic factors.

## 📊 Dataset
- 2,392 student records
- Features: Study time, Absences, Tutoring, Parental Support, etc.
- Source: Kaggle

## 🛠️ Tech Stack
- Python, Pandas, Scikit-learn
- Seaborn, Matplotlib (visualization)
- Random Forest Classifier

## 📈 Results
- **Accuracy: 91.4%**
- Strong precision/recall across most grade classes

## 🔍 Key Insights
- GPA and Absences are the top 2 factors affecting grade class (70%+ importance)
- Extracurricular activities have minimal impact
